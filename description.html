Introduction

The Blend for Visual Studio memory game sample project contains the files required to begin the tutorial described in "Design your first Metro style app built using HTML." The tutorial demonstrates how Blend for Visual Studio 2012 Release Candidate can be used to visually design a Metro style app, including using Interactive Mode to create styles for states that are only reachable through user interaction with the app.

Building the Sample

After you download the memory project, unzip the folder, and extract all of the files.
Browse to the directory in which you unzipped the sample. Open the memory folder and double-click memory.sln.

Description

This tutorial creates a lightweight, dynamic version of the memory game commonly referred to as "Concentration." In the game, a number of cards are laid face down. The objective of the game is to turn over pairs of matching cards until all of the matching pairs have been revealed.

The tutorial is based on a project that began with a developer who, working in Visual Studio 2012 Express Release Candidate, defined the sematic structure as HTML markup, and game logic developed using JavaScript.

The task in this tutorial is to use the design surface of Blend to complete the layout and styling tasks for a simple game app.
